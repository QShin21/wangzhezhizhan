# 王者之战身份模式

新月杀《三国杀王者之战》身份模式拓展包。

## 规则概要

- 仅允许 6 人或 8 人开局。
- 6 人身份：1 主公、1 忠臣、1 内奸、3 反贼。
- 8 人身份：1 主公、2 忠臣、1 内奸、4 反贼。
- 选将固定 60 秒，不受房间通常操作时长影响。
- 将池固定使用 `wangzhezhizhanwujiang` 武将包。
- 主公获得 4 名常备主和 3 名非主公将；其他身份获得 6 名非主公将。
- 游戏开始时，主公随机获得一张四象标记。
- 8 人局第三轮后、6 人局第四轮后进入鏖战阶段。
- 结算时按赛事手册计算积分，并在结算表称号栏补充概览、积分、击杀对象和死亡来源。
