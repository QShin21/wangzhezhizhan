# 王者之战身份模式

新月杀《三国杀王者之战》身份模式拓展包。

## 规则概要

- 仅允许 6 人或 8 人开局。
- 6 人身份：1 主公、1 忠臣、1 内奸、3 反贼。
- 8 人身份：1 主公、2 忠臣、1 内奸、4 反贼。
- 选将固定 60 秒，并限定使用 `wangzhezhizhanwujiang` 武将包。
- 主公从 4 名常备主和 3 名非主公武将中选择；其他身份从 6 名非主公武将中选择。
- 游戏开始时主公获得一张随机选择的四象标记。
- 8 人第三轮后、6 人第四轮后进入鏖战阶段。

> 计分面板需要客户端结算 UI 支持。本仓库记录击杀/胜负所需的模式身份与死亡奖惩逻辑，后续可按赛事手册补充 UI 结算表。
